# APK Output Folder

This folder will contain the built APK files.

## How to Build APK

See `BUILD_APK_GUIDE.md` in the project root for detailed instructions.

### Quick Start:
1. Install Android Studio
2. Open `android` folder in Android Studio
3. Build â†’ Build APK
4. Copy the APK from `android/app/build/outputs/apk/debug/` to this folder

## Files

- `dia-block-game.apk` - Debug APK (for testing)
- `dia-block-game-release.apk` - Release APK (for distribution)

## Installing on Your Phone

1. Enable Developer Options (tap Build Number 7 times in About Phone)
2. Enable USB Debugging
3. Transfer APK to phone
4. Open file manager and tap APK to install
